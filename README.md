# Design Cursor

Design Cursor 是一个面向 Chrome / Edge 的浏览器扩展，用来在真实网页里直接选中元素、记录结构和样式上下文，并生成适合粘贴到 Codex、Cursor、ChatGPT 等 AI 工具中的页面说明。

当前正式版本：`1.0.0`

## 它解决什么问题

和 AI 协作改 UI 时，最常见的问题不是“不会描述需求”，而是 AI 拿不到足够真实的页面上下文。

Design Cursor 把这件事做成了一个浏览器内工作流：

1. 在网页中直接圈出你想修改的元素
2. 自动提取文本、结构、布局和关键样式信息
3. 为元素补充你的修改要求或调整意图
4. 一键复制成结构化提示词，继续发给 AI 修改页面

它不是截图工具，而是一个面向 AI 工作流的“网页上下文采集器”。

## 1.0.0 版本能力

- 支持在网页中选中单个或多个 DOM 元素
- 支持选中文本片段，而不只是整个元素
- 自动整理页面标题、URL、选择器、文本、尺寸、位置、关键样式和 HTML 片段
- 支持为每个选中项补充单独的修改提示词
- 支持在清单中直接编辑每一项的内容要求
- 提供排版/调整模式，用于表达顺序调整、位置交换和样式修改
- 支持直接调整常见视觉属性，例如尺寸、布局方向、间距、内边距、填充、透明度、圆角和阴影
- 支持撤销 / 重做，方便反复试验
- 支持一键复制完整上下文到剪贴板

## 使用场景

- 让 AI 精准修改按钮、输入框、卡片、标题、段落等界面元素
- 一次收集多个相关元素，让 AI 做联动调整
- 表达“把这两个模块换位置”“把这一组内容重新排序”这类布局任务
- 在不依赖截图的前提下，把真实 DOM 上下文交给 AI 继续改页面

## 安装

1. 打开 Chrome 或 Edge 扩展页：`chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前项目目录

## 快速开始

1. 打开你想修改的网页
2. 点击扩展图标，或使用快捷键启动

```text
Mac: Command+Shift+Y
Windows: Ctrl+Shift+Y
```

3. 在页面中切换所需模式：浏览、输入、排版、调整
4. 选中目标元素或文本片段
5. 在清单里补充你的修改要求，或直接做排版/样式调整
6. 点击“复制全部”，把整理好的上下文粘贴到 AI 编辑器继续协作

更完整的功能说明见：[使用说明书.md](./使用说明书.md)

## 权限说明

- `activeTab`：仅在你点击扩展图标或使用快捷键后，对当前页面生效
- `clipboardWrite`：仅在你主动执行复制时写入剪贴板
- `scripting`：用于把扩展的内容脚本和样式注入当前页面，以运行元素选择器

## 隐私

Design Cursor 不依赖后端服务，不会把选中的页面内容发送到开发者服务器或第三方服务。

详细说明见：[PRIVACY_POLICY.md](./PRIVACY_POLICY.md)
