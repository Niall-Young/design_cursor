# Design cursor

Design cursor 是一个面向 Chrome / Edge 的浏览器扩展，用来在网页中直接选中元素，并把它们整理成适合粘贴到 AI 对话中的结构化上下文，方便做 UI 修改、布局调整和页面还原。

当前版本：`0.2.0`

## 项目定位

这个项目主要解决的是这样的工作流：

1. 在网页上直接圈出你想修改的元素
2. 把这些元素的结构、文本、尺寸、样式和位置信息整理出来
3. 复制到 Codex、Cursor、ChatGPT 或其他 AI 编辑器
4. 让 AI 基于真实页面上下文继续修改界面

它更像一个“网页设计上下文采集器”，而不是传统意义上的截图工具。

## 主要功能

- 直接在网页中选择一个或多个元素
- 支持选择文本片段或普通 DOM 元素
- 在工具栏面板里维护当前选择列表
- 复制适合发送给 AI 的结构化页面上下文
- 自动附带以下信息：
  - 页面标题
  - 页面地址
  - 元素选择器
  - 文本内容
  - 位置和尺寸
  - 关键计算样式
  - HTML 片段
- 支持给已选元素补充“修改提示词”
- 清单中可直接编辑每个条目的“内容要求”
- 支持布局调整模式
- 调整模式生成的样式要求会显示在清单中，并在复制时一并写入提示词
- 在布局调整模式下可多选元素并通过拖拽中心控制点交换位置
- 在布局调整模式下复制内容时，会额外标注这是“换位置 / 调整顺序”的任务

## 安装方式

1. 打开 Chrome 或 Edge 的扩展页：

```text
chrome://extensions/
```

2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前项目目录

## 使用方式

1. 打开你想修改的网页
2. 点击扩展图标，或使用快捷键启动：

```text
Mac: Command+Shift+Y
Windows: Ctrl+Shift+Y
```

3. 在页面中切换需要的模式：
   - 浏览模式
   - 选择模式
   - 布局调整模式
4. 在选择模式下，点击元素或文本，把它加入选择列表
5. 如有需要，可为某个已选元素补充修改提示词
6. 也可以在清单中直接编辑某个条目的内容要求，或查看调整模式生成的样式要求
7. 在布局调整模式下，可选择多个元素，并拖拽中心圆点来交换它们的位置
8. 点击“复制全部”把当前上下文复制到剪贴板
9. 将内容粘贴到 AI 编辑器中，继续描述你的修改需求

## 适合的使用场景

- 告诉 AI 修改某个按钮、输入框、卡片或文本样式
- 收集多个相关元素，让 AI 一次性做联动调整
- 把当前页面的真实 DOM 信息整理给 AI，而不是只给截图
- 在布局模式下表达“这两个元素换位置”或“把这一组内容重新排序”
