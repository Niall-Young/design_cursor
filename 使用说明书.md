# Design Cursor 使用说明书

适用版本：`1.0.0`

## 1. 这是什么

Design Cursor 是一个 Chrome / Edge 扩展，用来在真实网页中直接选中元素或文本，记录页面结构与样式信息，并生成适合粘贴到 Codex、Cursor、ChatGPT 等 AI 工具中的结构化上下文。

它适合这几类任务：

- 让 AI 精准修改某个按钮、标题、卡片、输入框或图片
- 一次收集多个相关元素，让 AI 做联动修改
- 表达“把这两个模块换位置”“把这组内容重新排一下”这类布局任务
- 在不依赖截图的情况下，把真实 DOM 信息交给 AI

## 2. 安装与启动

### 安装

1. 打开浏览器扩展页：`chrome://extensions/`
2. 打开“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前项目目录

### 启动方式

- 点击浏览器工具栏里的扩展图标
- 或使用快捷键：

```text
Mac: Command+Shift+Y
Windows: Ctrl+Shift+Y
```

再次点击图标或再次使用快捷键，可以重新切换扩展状态。

## 3. 界面说明

启动后，页面右下角会出现一组悬浮工具栏，主要由这些部分组成：

- `浏览模式`：暂停选择操作，只保留工具栏
- `调整模式`：直接修改元素样式，并把调整结果整理成说明
- `排版模式`：移动元素、交换顺序、复制元素到相邻位置
- `输入模式`：选中元素或文本，并为每个目标补充修改要求
- `选择列表`：查看已选目标、删除条目、复制单条、复制全部
- `撤销 / 重做`：回退或恢复选择、布局、样式调整
- `关闭`：退出当前页面的 Design Cursor

页面中被选中的目标旁边还会出现数字徽标：

- 在 `输入模式` 下，点击数字徽标可以编辑这条目标的文字修改要求
- 在 `调整模式` 下，点击数字徽标可以直接打开该元素的样式调整面板

## 4. 推荐工作流

最常见的用法如下：

1. 打开目标网页并启动 Design Cursor
2. 根据任务选择模式
3. 点击页面中的目标元素或文本
4. 按需补充文字要求、调整样式或拖动布局
5. 打开选择列表检查内容
6. 点击 `复制全部`，把结果粘贴给 AI

## 5. 四种模式怎么用

### 5.1 浏览模式

浏览模式相当于“安全模式”：

- 不会继续选中新元素
- 不会高亮悬停目标
- 已选内容会保留
- 适合临时查看页面、滚动页面、避免误操作

如果你已经选好了内容，但还不想继续编辑，可以切到浏览模式。

### 5.2 输入模式

输入模式用来收集目标，并为每个目标写“我要 AI 怎么改”。

### 适合什么场景

- 改按钮文案
- 改标题、段落、标签文本
- 改输入框提示词
- 改某个组件的文案、边框、颜色、尺寸等

### 基本操作

- 点击页面中的元素，可以添加或取消选择
- 如果点击到的是文本区域，扩展会尽量优先记录文本片段，而不是整块元素
- 如果元素结构贴得很紧，可以先选中一个子元素，再按 `Shift+Enter` 逐级选择父元素，按 `Enter` 返回子元素
- 选中后会出现数字徽标，点击它可以打开“添加修改”面板
- 面板里可以直接输入要求，例如“修改文案为立即开始”“修改字号为 18px”
- 面板下方有快捷短语，点一下就能插入常见指令

### 保存规则

- 点击面板外部：自动保存当前输入
- 按 `Esc`：关闭当前输入面板，并放弃这次未保存的修改

### 5.3 调整模式

调整模式会直接对当前网页元素施加样式修改，并把这些改动同步整理为“调整模式要求”。

### 基本操作

- 点击一个元素，会把它加入选择列表并立即打开调整面板
- 调整模式只处理整块元素，不处理文本片段
- 如果要调整外层容器但只能点中里面的区块，可以先点中子元素，再用 `Shift+Enter` 上钻到父级容器
- 修改会先直接预览在当前网页上
- 关闭调整面板后，改动会保留；如果需要回退，可以用撤销

### 可调整的内容

#### 布局

- 自动布局方向：关闭 / 横向 / 纵向
- 宽高：`Hug`（自适应） / `Fixed`（固定） / `Fill`（填满）
- 对齐：九宫格对齐
- 间距：`gap`
- 内边距：上、右、下、左

#### 外观

- 透明度
- 圆角

#### 颜色

- 基础填充颜色
- 新增叠加颜色层
- 切换填充显隐
- 删除填充层
- 实色 / 渐变切换
- 渐变角度
- 新增渐变节点
- 调换渐变方向
- 取色器吸色
- Hex / RGB / HSL 输入格式
- Alpha 透明度

#### 阴影

- 外阴影
- 内阴影
- 额外新增阴影层
- 阴影 X / Y / Blur
- 阴影颜色与透明度
- 切换阴影显隐
- 删除阴影

### 数值拖拽

在调整面板里，很多数字输入框支持快速拖拽：

- 按住 `Alt`
- 在数字输入框上左右拖动
- 数值会连续变化，适合快速试圆角、透明度、间距、宽高等参数

### 图层复制与粘贴

颜色层和阴影层支持复制粘贴，适合把同一套效果快速套到别的元素上。

- 在颜色行或阴影行的左右边缘区域点击，可以选中这一层
- 按住 `Shift` 再点，可以多选同类图层
- 按 `Command/Ctrl+C` 复制已选图层
- 切到另一个元素后，按 `Command/Ctrl+V` 粘贴图层

### 5.4 排版模式

排版模式适合处理“位置”和“顺序”问题，而不是单纯改颜色或文案。

### 适合什么场景

- 调整两个模块的先后顺序
- 把一个元素拖到另一个位置
- 复制一个卡片到左侧、右侧、上方或下方
- 让 AI 理解“这里要重新排版”

### 基本操作

- 点击元素可以加入或取消选择
- 进入排版模式后，只保留元素目标，不保留文本片段
- 面对嵌套很深的卡片或弹窗，可以先选中内部元素，再用 `Shift+Enter / Enter` 在父子层级之间切换目标
- 已选中的元素中间会出现拖拽手柄
- 拖动手柄可以移动位置
- 拖到其他元素附近时，可能会触发插入或交换位置

### 复制相邻元素

当鼠标悬停在已选元素边缘附近时，会出现 `+` 按钮：

- 横向布局里通常会显示在左侧或右侧
- 纵向布局里通常会显示在上方或下方
- 点击后会复制当前元素，并插入到该方向相邻位置

### 自动生成的排版说明

在排版模式中发生拖拽、交换、复制后，扩展会自动为目标补充排版说明，例如：

- 把某个元素移动到另一元素前后
- 交换两个元素的位置
- 复制一个元素到原元素左侧、右侧、上方或下方

这样在复制给 AI 时，对方能更容易理解这是“布局调整任务”。

## 6. 选择列表怎么用

点击工具栏里的列表按钮，可以打开“调整清单”。

列表里可以做这些事：

- 查看当前一共选中了多少个目标
- 查看每条目标的大致标签、尺寸和选择器
- 看到这条目标是否包含 `内容变动 / 排版变动 / 调整变动`
- 删除单条目标
- 单独复制某一条目标
- 清空全部选择
- 一次复制全部目标

如果当前没有任何选择，列表会显示空状态。

## 7. 复制出来的内容包含什么

点击 `复制全部` 后，复制到剪贴板的内容通常会包含：

- 页面标题
- 页面地址
- 当前选择模式
- 每个目标的类型、标签描述、选择器、角色、Aria 标签
- 文本内容
- 尺寸和位置
- 关键样式
- HTML 片段
- 你补充的内容要求 / 排版要求 / 调整要求

如果当前任务属于排版模式或调整模式，还会自动附带对应的全局说明，帮助 AI 判断任务类型。

## 8. 快捷键与操作习惯

- `Command+Shift+Y / Ctrl+Shift+Y`：打开或切换 Design Cursor
- `Esc`：按当前上下文逐级关闭
- 关闭顺序通常是：输入面板 / 调整面板 / 选择列表 / 整个扩展
- `Shift+Enter`：当前单选元素时，选择最近的可选父元素
- `Enter`：当前单选元素时，选择优先匹配的子元素
- `Command/Ctrl+C`：有已选目标时，直接复制当前上下文
- `Command/Ctrl+Z`：撤销
- `Shift+Command/Ctrl+Z`：重做
- `Alt + 拖动数字输入框`：快速调整数值

## 9. 常见使用方式

### 只改一个按钮文案

1. 进入 `输入模式`
2. 点击按钮
3. 点击数字徽标
4. 输入“修改按钮文案为立即预约”
5. 复制全部并发给 AI

### 调整一个卡片的视觉样式

1. 进入 `调整模式`
2. 点击卡片
3. 修改圆角、阴影、填充、内边距等参数
4. 复制全部并发给 AI

### 重新排列页面模块

1. 进入 `排版模式`
2. 选中需要调整的模块
3. 直接拖到新位置，或用边缘 `+` 按钮复制相邻模块
4. 复制全部并发给 AI

## 10. 注意事项

- 扩展只能在普通网页中运行，不能注入浏览器内部页面
- `浏览模式` 只保留工具栏，不做选择操作
- `输入模式` 更适合“写要求”，`调整模式` 更适合“直接改样式”
- `排版模式` 只处理元素，不处理文本片段
- 输入面板点外部会保存，按 `Esc` 会取消当前未保存输入
- 该扩展没有后端，不会自动把页面内容发送到开发者服务器

## 11. 一句话总结

如果你想告诉 AI “改什么”，用 `输入模式`；如果你想先在网页上“试着改”，用 `调整模式`；如果你想表达“位置怎么变”，用 `排版模式`；如果你只是想暂停误操作，用 `浏览模式`。
